# tests/0-add_integer.txt

``add_integer()`` returns the addition of is arguments. For
integers and floats, that value is equivalent to using the ``+``
operator

::
	>>> add_integer = __import__('0-add_integer').add_integer
	>>> add_integer(8, 10)
	18
	>>> add_integer(8, 20)
	28
	>>> add_integer(8, 30)
	38
	>>> add_integer(8, 40)
	48
	>>> add_integer(8, 50)
	58
	>>> add_integer(8, 60)
	68
	>>> add_integer(8, 70)
	78
	>>> add_integer(8, 80)
	88
